# Desktop Application Guide

Using the CV-Mindcare CustomTkinter GUI launcher.

## Starting the Desktop App

```bash
python launcher/main.py
```

Or with the installed script:
```bash
cv-mindcare
```

## Features

- System tray integration
- Start/stop sensors with one click
- View real-time sensor data
- Configure settings
- Process management

## Interface

The desktop app provides:
1. **Main Window**: Sensor status and controls
2. **System Tray**: Quick access and notifications
3. **Settings Dialog**: Configuration management
4. **Logs Viewer**: Application logs

See [Features Guide](features.md) for complete details.
